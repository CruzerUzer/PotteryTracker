# PotteryTracker Visual Mockups

This directory contains interactive HTML mockups that you can open in your browser to see the visual designs.

## Files

### Mobile Mockups
- **mobile-light-kanban.html** - Mobile Kanban view in light theme
- **mobile-dark-kanban.html** - Mobile Kanban view in dark theme
- **mobile-light-detail.html** - Mobile piece detail view in light theme

### Desktop Mockups
- **desktop-light-kanban.html** - Desktop Kanban view in light theme
- **desktop-dark-kanban.html** - Desktop Kanban view in dark theme
- **desktop-light-detail.html** - Desktop piece detail view in light theme

## How to View

1. Open any HTML file in your web browser
2. The mockups are sized appropriately for their device type:
   - Mobile: 375px wide (iPhone size)
   - Desktop: Full width with sidebar navigation

## Features Shown

- **Color Schemes**: Light and dark themes with exact color values
- **Layouts**: Responsive designs for mobile and desktop
- **Components**: Cards, buttons, navigation, forms
- **Typography**: Font sizes, weights, and spacing
- **Interactions**: Hover states, active states

## Notes

- These are static mockups (no JavaScript functionality)
- Colors match the design proposal specifications
- Layouts are pixel-perfect representations
- All components use the proposed design system

## Next Steps

These mockups can be used as reference during implementation to ensure the final design matches the vision.
